# Space Explorer

School project in Unity

## Presentation

### Description

A Asteroids-like game in which the player controls a spaceship and must dodge asteroids to survive.

### Features

- Movements with Arrows keys or ZQSD
- Map
  - 1 playable map
  - Randomly generated asteroids
- Menus
  - Main menu
  - Pause menu
  - End menu

### Technical

- Unity 2022.3.9f1
- C#
- 2D
