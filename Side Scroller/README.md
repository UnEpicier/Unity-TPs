# Side Scroller

> Vasseur Alexis

## Target

Make a infinite _side scroller_.

The player must jump on randomly generated platforms to avoid falling into the void.
Each successful jump is worth one point.

## Content

- Player
- Platform Spawner
  - Platform
- Random Background
- UI
  - Start Menu
  - Countdown screen
  - Overlay (Display score)
  - Loose screen
  - Pause Menu
